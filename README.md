💜 Quiz BTS 💜

Um quiz interativo feito com Python e Streamlit sobre o BTS.

 Acesse o projeto online!

 Link do App: (https://quizbts-stizunpqwowffnxxmjeiw3.streamlit.app/)

🧠 Funcionalidades
	•	Perguntas sobre BTS
	•	Sistema de pontuação
	•	Classificação (iniciante, intermediário, raiz)
	•	Exibição de imagem no final

- Tecnologias usadas
	•	Python
	•	Streamlit
	•	Pillow

- Como rodar localmente
pip install -r requirements.txt
streamlit run app.py
